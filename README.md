# A-Self-Adaptive-MOPSO-Using-Fractional-Velocity-Memory-and-Mittag-Leffler-Exploration
Implementation of SA-MOPSO-FVML, a self-adaptive multi-objective PSO with fractional-order velocity memory and Mittag-Leffler exploration, validated on ZDT/DTLZ benchmarks and applied to marine vessel dynamic positioning under realistic environmental constraints.
- multiObjectives.m: Benchmark multi-objective test functions (ZDT, DTLZ)(run this file)
- MiniAblation.m: Ablation-based comparative analysis(run this file)
